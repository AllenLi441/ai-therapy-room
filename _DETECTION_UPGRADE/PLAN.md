# 静室危险识别升级 — 计划表（设计 workflow 产出,2026-06）

## 核心反框（必须先达成共识）
"识别所有倾向" 字面 = 召回=1.0 → 在子串/正则词表上把精度压到地板 → 正常倾诉(吐槽/丧亲/惊恐/存在性思考,占用户绝大多数)被砸进危机锁定 → app 变成**医源性伤害**、没人回来。
**真正目标:分级、非对称、按严重度加权的召回**
- 只对**最高危、最低基率**的倾向(预演/中断未遂、既往未遂复发、具体手段+计划、围产期母婴、扩大自杀)放宽精度换召回——这里误报只是软关心,漏报是灾难。
- 对**高基率模糊**倾向(被动意念、纯困顿、夸张、丧亲)**死守精度**——这里淹没多数人是主要伤害。
- 召回靠**分级路由**买,不靠危机升级:检测到 → 软澄清;只有确认/高危才进完整危机模板。
- 每一点召回提升都要用**该类目的精度预算**付账:逐类召回↑ **且** 该类目 FP 在冻结负样本集上不破上限,才能合并。

## 关键发现(来自 9-agent 只读审计)
- **基建洞 #1**:`-flash` 模式跳过整个 LLM 语义层(route.ts:618 先 return)。
- **基建洞 #2**:Kimi 报错 + 词表 none → `fail_safe_release` 静默放行(implicit-risk.ts:699)。
- **词表洞**:`normalizeText`(safety.ts:608)只做小写+去空格 → **繁体全漏**(自殺/結束生命/燒炭/遺書)、全角漏、**插字符规避**(自-杀/自。杀)、谐音/拆字(自sha/zs/4了/寄了/嘎了)、同义/手段超出固定枚举(卧轨/触电/高空/未列药名)。
- **LLM 洞**:无纯英文 few-shot(英文伪装/诗化无法泛化);`hasActionableImplicitEvidence` 关键词门 + conf≥0.4 地板会丢真阳;8 轮/600 字窗口外的梯度丢失;`pragmatic!==self` 绝对放行(自被误标成 other 就漏)。
- **守卫洞**:口头否认抹掉真实 MEDIUM 被动意念(carve-out 只保护 HIGH OD/LD/means);4 轮拼接里旧否认毒化后续新意念;benign-anchor 游戏化(告别+平静混过累加器);**降级语+手边有药同轮 → 释放危机**(我 Fix B 只修了一半,缺 means 感知)。
- **eval 洞(最致命)**:**从不计算召回/精度/F1**,只有 pass/total;CI 门只测词表层(/api/safety-check 从不调 LLM);10000/10000 是 ~150 个固定词干撑出来的幻觉;无逐类召回;标签自产(codex-template-v1,循环验证);无临床 ground truth。

## 不可交易的护栏
1. **eval-first**:逐类召回+精度 harness + 冻结 FP 门**先于任何词表改动**存在。
2. **FP 回归门不可交易**:任一良性类目 FP 破上限即 CI 失败,哪怕召回↑。裸高频词(suicide/dead/kill/消失/困住/unalive/kms)**禁止进硬 HIGH 词表**,只能作累加器/LLM 层信号。
3. **检测 ≠ 完整危机**:分级路由先行。
4. **基建洞先修**:flash 旁路 + Kimi 放行不修,词表白加。
5. **折叠必重跑成语回归**。
6. **守卫交互锁成回归 fixture**(否认抹除/旧否认毒化/benign-anchor/降级+手边药)。
7. **shadow/canary 先行**:召回改动先影子模式跑 prod 遥测,FP 在预算内才上线。
8. **临床 ground truth**:理想是持证临床签字。**现实(你单人项目无临床)→ 适配:你逐批审 + 保守分级路由 + FP 门作安全网**;对外措辞用"分层风险筛查",绝不说"识别一切"。

## DO-NOT
- 别给硬 HIGH 词表加裸高频词(子串无词界 → "自杀热线"/"饿死了"/"真想跳楼甲方改8版" 全中)。
- 别在 harness(逐类召回+FP 门)就位前 ship 任何召回改动。
- 别把检测等同完整危机模板(对丧亲/惊恐/存在性思考多数是医源伤害)。
- flash 旁路 + Kimi 放行没修前,别追词表"完整性"。
- 繁→简/NFKC 不重跑成语回归别上(乾/幹→干 等新碰撞)。
- 别全局降 conf 地板/去 cue 门(重新放进被压制的 FP);只能按 tier 定向恢复。
- 别为精度把否认/降级判得更绝对(没有守卫回归 fixture 绿)。
- 别靠加更多 opener 扩"假如有个人"伪装升级(破坏真三方关心/创作/学术);判别子必须是"具体可映射到第一人称的处境+手段+撇清"。
- 别在自产标签上 ship 或对外宣称"识别一切"。
- **别在这台 M4 上铺重并行**(不同时 dev+build+test;验证用一次性 build;不留后台 dev server)。
