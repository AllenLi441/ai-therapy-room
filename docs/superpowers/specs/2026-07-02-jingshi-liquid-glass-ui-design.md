# 静室 UI 全面重设计 — Liquid Glass(苹果风)spec

- 日期: 2026-07-02 · 分支: `p6-ui-redesign`(基于 `p5-clinical-rag`)
- 方向由用户拍板: **苹果风 + 液态玻璃**;审美细节全权委托。
- 流程: 直接写码,节点截图给用户确认;安全组件全保留。

## 1. 论点

静室不是聊天框,是一间有光的房间。本次以 Apple 2025 Liquid Glass 语言重做整个前端:
**流动的环境光场在最底层,一切界面元素是浮在光上的玻璃。** 安屿不是气泡里的客服,
是房间里带柔光的存在(Siri-glow)。

## 2. 技术底座(吸取 Mineradio 官网教训)

- **不整包使用 `liquid-glass-react`**(Safari 不渲染折射)。玻璃 = 纯 CSS:
  `backdrop-filter: blur(Xpx) saturate(1.6)` + 多层 tint + 1px 高光描边 +
  顶缘 specular 内渐变 + 阴影梯度 —— Safari/微信内嵌全兼容。
- Chromium 增强: SVG displacement 折射(feature-detect,`@supports` + UA 探测),
  降级无感。
- 字体: 纯系统栈 `-apple-system, "SF Pro", "PingFang SC", "MiSans", "Noto Sans SC"`,
  零字体下载;数字用 `font-variant-numeric: tabular-nums`(热线号码加大)。
- 环境光场: 固定层 CSS 渐变斑(2-3 个 blob,transform 动画 ~30s 周期,GPU 便宜),
  `prefers-reduced-motion` 时静止。
- 动效: 弹层 spring 呈现、悬浮岛微视差、安屿呼吸柔光(~8s);reduced-motion 全部关闭。

## 3. 设计 tokens

### 光场(底层)
- 日房: 纸光 `#F4F2ED` 底 + 晨光斑 `#F6E7D3` + 青瓷斑 `#9DBBAA`(延续品牌绿)+ 淡天青 `#C7D9D4`
- 夜房: 深青墨 `#12171A` 底 + 暖灯斑 `#3A2F22` + 夜青斑 `#1E3330` + 微月光 `#2A3540`
- 主题: 日/夜双态,默认跟系统 `prefers-color-scheme`,手动可切(沿用 `js_theme` 键)。

### 玻璃(表层)
- `--glass-blur: 24px; --glass-sat: 1.6`
- 日: tint `rgba(255,255,255,.55)`,描边 `rgba(255,255,255,.65)`,阴影 `0 8px 32px rgba(31,38,27,.12)`
- 夜: tint `rgba(22,28,30,.5)`,描边 `rgba(255,255,255,.14)`,顶缘 specular `rgba(255,255,255,.25)`
- 圆角语言: 卡 24-28px、岛 32px、pill 999(Apple continuous-corner 视感)。

### 语义色
- 品牌青瓷 `#7FA08F`(日)/ `#8FB3A2`(夜)— 安屿标记、链接、选中态
- 安全朱 `#C4553D`(日)/ `#D96A50`(夜)— **仅**危机/急救语义,永不装饰
- 文字: 日 `#1E2422` / 次 `#5A6660`;夜 `#EDEAE3` / 次 `#9AA6A0`

### 排版(中文优先)
- 正文 17px / 行高 1.9;用户消息 15.5px;工具文字 12.5px 加宽字距
- 安屿回复直接排在光场上,阅读列宽 ~34em;显示层级靠字重(500/600)不靠花字体

## 4. 组件重做清单(逻辑与接线不动)

| 组件 | 玻璃化方案 |
|---|---|
| 顶栏 | 极薄玻璃条,滚动时才显材质;左 wordmark,右 日/夜·语言·关于 |
| 隐私条 | 并入写字台上方一行小字,不再永久占一条横幅 |
| 欢迎屏 | 光场缓亮 + 安屿柔光呈现 + 现有文案渐显;开场引导 = 无边框玻璃行 |
| 对话流 | 用户 = 右侧玻璃染色气泡;安屿 = 无气泡文字 + 左侧呼吸柔光点 |
| 思考过程 | 折叠玻璃笺(磨砂更重、字更小),点开展开 |
| 信息来源 | 回复下缘一排玻璃引条,展开为笺注(URL+逐字引用原样) |
| 危机横幅/卡 | **最高层级玻璃**:暖色 tint+朱色文字锚点,光场同时沉静(dim);热线数字 tabular 加大;文案一字不改 |
| 写字台(composer) | 底部浮岛(边缘内缩,iOS 26 式),+ 图片、深度/快速在岛内;发送 = 品牌青瓷实心圆 |
| 弹层(量表/呼吸/档案/关于) | 玻璃 sheet,spring 呈现;呼吸练习与光场联动(吸 4s 亮/呼 6s 暗) |
| 一键彻底删除 | 保留确认弹层,玻璃化;红色动作用安全朱 |

## 5. 不变量

- 全部安全组件功能与**临床审定文案逐字保留**(危机卡、热线 110/120/12356、免责声明、
  量表、删除确认);crisis 触发逻辑(X-Crisis-Triggered)不动。
- 不出现「临床评审/已审核」类表述。
- 流式协议、X-Knowledge 来源、思考面板数据流、/api 接线全部不动(只改呈现层)。
- zh/en 双语、light/dark、`prefers-reduced-motion`、移动端(safe-area)全覆盖。
- 无障碍: 焦点可见、对比度达标(玻璃上文字加 text-shadow/加深 tint 保底)。
