# 人工标注研究协议(v0.1 草案 · 2026-07-13 · 待联合复审定稿)

> 目的:用**人类金标**替代模型互评,终结「裁判=选手」问题(金标 LLM 生成、
> annotator_A=DeepSeek、annotator_B=Kimi、判官臂也是这两家——每层打分者都是参赛者)。
> **预注册预期:判官臂在模型金标上的 97.0% 二值召回,在人工金标上预期回落。**
> 回落本身就是论文发现,不是失败。

## 1. 标注者

- **2–3 名心理学背景学生**(修过变态心理学/临床方向优先),或由**心理学教师签名背书**;
- 相互独立、盲标:看不到彼此标注、看不到模型任何输出、看不到种子原始标签;
- 标注前须通读 `eval/datasets/safety/ANNOTATION_GUIDE.md`(四级定义对齐 C-SSRS,已存在,不另造)。

## 2. 标注对象与抽样

- 总池:344 个标注单元(256 条种子按用户轮展开);**多轮按整条对话抽取、逐轮标注**(不拆散上下文;统计时声明簇内相关使有效 n 略小);
- **主集 100 单元**,四类均衡(全池实测 none 177 / passive 60 / active 60 / crisis 47):

| 层 | 全池 | 抽样 | 说明 |
|---|---|---|---|
| none | 177 | 25 | 从 9 个 category 分层,优先 hard_negative_slang / negation / recovery_statement(双模型标注最不可靠处:recovery κ=0.07) |
| passive_ideation | 60 | 25 | 最弱检测环,重点层 |
| active_ideation | 60 | 25 | |
| crisis | 47 | 25 | |

- **副集(推荐)**:A/B 双模型标注的全部分歧单元(约 74 条,与主集重叠部分不重复标)——只用于数据质量审计与人工排查优先级,**不进 κ 与主统计**(分歧过采样会压 κ);
- 另抽 **10 条校准集**(不计入正式统计),呈现顺序全局随机、去掉任何暗示标签的字段;
- 每条增加一个辅助标注字段:**「文本不自然/不可标」**——顺带完成合成数据真实性审计(审稿人必问);
- 均衡抽样后报告:分层数字 + 按全池权重回推的整体估计,两者都给;
- **统计功效边界(如实写进论文)**:每类 n≈25 时单类召回 95%CI 半宽约 ±13–19pp——足以验证「判官 97% vs 管线 54–66%」这类悬殊差,**不足以分辨 <15pp 的臂间差**。

## 3. 金标规则(用户定):交叉验证

- 三人:**2/3 多数 = 人工金标**;三人全歧 → 教师仲裁定标;
- 两人:一致 = 金标;不一致条目 → 教师仲裁(并在论文中报告两人版的局限);
- 一致性指标:三人 Fleiss κ(两人则 Cohen κ),**目标 ≥ 0.6(Substantial)**;
- 校准轮 κ < 0.4 → 重训一轮(逐条讨论校准集分歧)再进正式集,校准讨论不许触及正式集条目。

## 4. 伦理与安全

- 全部文本为**合成样例**(`source:generated`),不含任何真实用户内容——明确告知标注者;
- 内容涉及自杀/自伤表达:知情同意书 + 可随时退出 + 附心理支持资源(12356);
- 标注数据只存本地,不外发(与仓库敏感数据边界一致)。

## 5. 产出与重打分

1. `eval/human-study/human_gold.jsonl`(id#turn → human_label + 每人原始标注 + κ);
2. **全部检测臂在人工金标子集上重打分**(judge / lexicon / pipeline_fast / pipeline_deep / baseline),
   与模型金标结果并排呈现——差值本身是「模型互评偏差」的量化证据(论文卖点之一);
3. 种子标签与人工金标的分歧清单 → 数据集 `human_review` 字段按条目推进。
